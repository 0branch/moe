## moe

moe is a Vim inspired command line based text editor.

This project's goal is a funny animation editor.

![moe-0](https://user-images.githubusercontent.com/15966436/54860567-7a43b400-4d5f-11e9-9740-8ab60b806c49.png)

## Features

- Written in Nim  

- Adopt GapBuffer  

- UTF-8 and other encodings support (Incomplete)

- Vim like mode (Normal, Insert, Visual, Replace, Ex, Filer)

- Vim like keybinds

- Syntax highlighting (Nim, C, C++, C#, Java, Yaml)

- Configuration file

- Simple file manager

- Auto close paren  

- Simple auto indent
