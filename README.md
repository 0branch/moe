## moe

moe is a command line based editor.

This project's goal is a funny animation editor.

![moe-0](https://user-images.githubusercontent.com/15966436/42743686-90deb5f8-88b5-11e8-9e5e-ae8202f64a98.jpg)

## Features
GapBuffer  

Written in nim  

Vim like key binds  

Simple file manager

Auto close paren  

Simple auto indent  

## How to Build
```sh
nimble build
```
